An AI slop version of Uiverse that will inevitably get deleted  
UIcommon is a library of UI components, made by Nodejs, React, Typescript and MongoDB 7.0.
backend env:
```
# Google OAuth
MONGO_URI=mongodb://localhost:27017/uicommons

GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=

GITHUB_CLIENT_ID=
GITHUB_CLIENT_SECRET=

DISCORD_CLIENT_ID=
DISCORD_CLIENT_SECRET=
# URL Backend của bạn

API_URL=http://localhost:3000

# URL Callback mà Google sẽ gọi về (BẮT BUỘC)

GOOGLE_CALLBACK_URL=http://localhost:3000/auth/google/redirect
GITHUB_CALLBACK_URL=http://localhost:3000/auth/github/redirect
DISCORD_CALLBACK_URL=http://localhost:3000/auth/discord/redirect
# JWT Secret

JWT_SECRET=
```
